hexo中插入图片
发表于2022-09-04
一丶下载插件 在博客根目录下git bash npm install hexo-asset-image –save 修改老版本的文件 node_modules –> hexo-asset-image –> index.js 记事本打开： $(this).attr(‘src’, config.root + link + src); 把这一行代码改成下面这样 $(this).attr(‘src’, src); 保存文件。二丶更改站点配置 打开站点配置文件并修改 _config.yml中的 post_asset_folder: false 改为true。三丶插入图片 在md编辑工具中使用md语法插入图片 and 在网页上预览 结束~注： 这样设置下来在主页无法显示图片，会有一个显示错误的图标很难看，这里可以用预览来解决 在文章编辑的头部加入 description：XXX 后面写上预览的部分，图片错误就会消失啦 给文章加上预览的功能方法之一就是这样~
